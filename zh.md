# ER Designer

专业实体关系图(ERD)数据库设计工具

[在 App Store 下载](https://apps.apple.com/app/er-designer/id6670524297?mt=12){: .btn }

## 主要功能

- **MySQL/PostgreSQL 支持**  
  全面支持主流数据库，智能解析表结构并实时生成 SQL 预览。

- **专业 ER 图设计**  
  清晰展示表间关联关系，帮助您掌握数据架构全局。

- **高效正向工程**  
  从可视化 ER 图一键生成精确的数据库结构和 SQL 代码。

- **智能逆向工程**  
  快速从现有数据库导入结构和关系，自动转换为直观 ER 图。
