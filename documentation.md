<h2 id="installation-guide">Installation Guide</h2>

1. Visit the [ER Designer App Store page]({{ site.appstore_link }})
2. Click the "Get" or price button to download
3. After installation, launch ER Designer from your Applications folder

<h2 id="quick-start-tutorial">Quick Start Tutorial</h2>

Follow this simple tutorial to create your first ER diagram:

1. Launch ER Designer
2. Click "New Project" to start a new diagram
3. Add your first table using the "Add Table" button
4. Define columns, keys, and data types
5. Create more tables as needed
6. Connect tables to define relationships
7. Generate SQL code using the "Generate SQL" option

<h2 id="user-interface-overview">User Interface Overview</h2>

ER Designer features an intuitive interface with:

- **Toolbar**: Quick access to common functions
- **Canvas**: Main diagram editing area
- **Properties Panel**: Detailed settings for selected elements
- **Navigator**: Overview and zoom controls for large diagrams

For detailed information about specific features, please refer to the corresponding section above.
